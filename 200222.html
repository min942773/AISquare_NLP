※ 자연어처리

  • 형태소 분석
  • 구문 분석
  • 의미 분석
  • 담화 분석

※ 절차

  1. 수집
  2. 탐색(EDA) : 해당하는 변수, 변수의 특성, 의미 등을 파악하는 단계
  3. 전처리(정제) : 가장 중요

※ 리눅스 명령어들

  1. conda update -n base conda
  2. conda upgrade --all
  3. conda create --name nlp anaconda : 가상환경 설정 (뒤에 아나콘다를 붙이면 현재 아나콘다에 설치되어있는 패키지를 그대로 가져옴)
  4. conda info --envs : 가상환경 목록
  5. conda activate nlp
  6. conda deactivate
  7. conda search selenium : 있으면 이걸로 설치, 없으면 pip로 설치

※ 수집

  • 수집 : 자연어 처리를 위한 Data 수집
  • file 형태 제공 : *.csv, *.txt, *.json, *.xml
  • web에서 필요한 data 수집 (web scraping / crawling)

※ web 구조

  • client가 request를 보내면(GET 방식 / POST 방식) server가 HTML 형태로 response 함
  • requests : HTML 수신
  • BeautifulSoup : HTML 분석
  • Selenium : Crawling 자동화

※ web의 특성

  1. 비연결 지향 (원래 tcp는 연결지향이지만 http는 반대)
  2. 상태값을 관리하지 않음 -> 이를 해결하기 위해서 cookie와 session 활용 ( cookie는 client쪽에서 관리, session은 server쪽에서 관리)

※ html

  • class : 중복 허용, css 적용 목적으로 많이 씀
  • id : 중복 비허용

※ web page

  • 구조 : html
  • 모양 : css
  • 동작 : JavaScript

※ crawling VS scraping

  • crawling : 필요한 것만 긁어오는 것
  • scraping : 웹 페이지 전체를 가져오는 것
